Drivecycle 0.1.0 Documentation
======================================


**Drivecycle** is a python package to simulate public transit bus trajectories.

.. note::

   This project is under active development.

Installation
------------

To use Drivecycle, first install it using pip:

.. code-block:: console

   (.venv) $ pip install
